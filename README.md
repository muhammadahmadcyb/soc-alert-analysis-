# soc-alert-analysis

## Description
This project demonstrates basic SOC operations including alert triage and incident categorization in a simulated lab environment.

## Tasks Performed
- Reviewed security alerts
- Identified phishing indicators
- Investigated brute-force login attempts
- Categorized alerts based on severity

## Tools Used
- SIEM dashboard (lab environment)
- Windows Event Logs

## Outcome
Improved understanding of alert investigation and incident documentation.
